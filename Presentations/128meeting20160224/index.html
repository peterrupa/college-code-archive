Features

It will be like Google Classroom. Sort of.

Our only target users: professors.

Each professors can have their own account.

Each account can have multiple classes.

Each class can have multiple activities and students.

Each student can have multiple tags, which can be used for filtering in the randomizer.

An activity is simply a session in which the professor will use their randomizer based on the students of that class.

Randomizer

There will be two pages: Control Panel and Presentation.

Control panel is where the professor can change the settings of the randomizer.

Presentation is the page in which the randomized names are displayed.
